EnglishAdvanced Large Ship Assembly ArrayA mobile assembly facility where advanced large ships such as Black Ops, Marauder class battleships as well as Jump Freighters can be manufactured. Note: To use a ship from a Ship Assembly Array a Ship Maintenance Array with enough free storage space needs to be in range that the ship can be moved there.
