EnglishMedium Ancillary Remote Shield BoosterTransfers shield power over to the target ship, aiding in its defense. The module takes Cap Booster charges and will start consuming the ship's capacitor upon the charges running out. Deactivating the module while it has no cap boosters loaded starts reloading, if there are cap boosters available in cargo hold. Note: Can use Cap Booster 50, 75 and 100 as fuel. Reloading time is 60 seconds. Prototype Inferno Module.

RussianMedium Ancillary Remote Shield BoosterДистанционно передаёт энергию силовым полям корабля-цели, тем самым повышая его защиту.
