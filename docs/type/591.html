GermanTormentorDie Tormentor ist bereits seit vielen Jahrzehnten im Einsatz. Die meiste Zeit diente sie als Bergbauschiff, da sie aufgrund ihrer Größe auf dem Schlachtfeld wenig ausrichten konnte.
