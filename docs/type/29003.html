GermanFocused Warp Disruption ScriptDiese Skripte wurden ursprünglich von einer jugendlichen Hackergruppe namens "The Fendahlian Collective" verwendet, um an High-Tech-Sicherheitsfirmware vorbeizukommen. Sie enthalten einen komplexen Code, der die inneren Abläufe bestimmter Modulen verändert, indem er direkte Befehle in die Firmware eingibt. Die Einsatzmöglichkeiten dieser Technologie blieben den großen Corporations nicht verborgen, und die Jugendlichen, die sie entwickelt hatten, leiten nun ihre eigenen Forschungsabteilungen. Dieses Skript kann in einen Warpstörfeldgenerator geladen werden, um seinen Effekt wie bei einem gewöhnlichen Warpstörer auf ein einzelnes Schiff zu lenken. Dadurch kann das Modul Schiffe jeder Größe stören, einschließlich solcher, die für gewöhnlich immun gegen alle Arten der elektronischen Kriegsführung sind.
FrenchScript de perturbation de warp concentréeInitialement développés par le groupe de pirates informatiques adolescents du « Collectif fendahlien » pour pénétrer illicitement dans des micrologiciels de sécurité de pointe, ces packs de scripts complexes permettent d'implanter des commandes directement dans le code de micrologiciels pour détourner le fonctionnement de certains modules.
