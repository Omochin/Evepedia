Warfare links are dedicated fleet command systems designed for use on battlecruisers and advanced command class ships. Notes: The Fleet bonus only works if you are the assigned fleet booster. This module cannot activate inside a starbase forcefield. GermanArmored Warfare Link - Damage Control ISenkt den Energiespeicherbedarf der persönlichen und gezielten Panzerungsreparatursysteme der Flotte.
